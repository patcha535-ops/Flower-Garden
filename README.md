# 🌸 Flower Garden Mobile (Puzzle Game)

เกมพัซเซิลแนววางแผนบนเว็บเบราว์เซอร์ที่เน้นความเพลิดเพลินและการใช้ความคิด พัฒนาด้วยเทคโนโลยีเว็บมาตรฐาน (HTML5, CSS3, JavaScript) โดยปรับปรุงให้รองรับการเล่นบนมือถือได้อย่างสมบูรณ์

## 🌟 ฟีเจอร์เด็ด (Key Features)
- **Wildcard System (Rainbow Flower 🌈):** เพิ่มความสนุกด้วยดอกไม้พิเศษที่สามารถใช้แทนดอกไม้สีใดก็ได้ เพื่อช่วยในการทำคะแนน
- **Mobile Responsive Design:** หน้าจอเกมปรับขนาดอัตโนมัติ (Responsive) ให้พอดีกับหน้าจอมือถือและแท็บเล็ต
- **Smooth Animations:** เอฟเฟกต์การปรากฏของดอกไม้แบบ Pop-up และการสั่นแจ้งเตือนเมื่อเลือกดอกไม้
- **Score System:** ระบบคำนวณคะแนนแบบ Real-time เมื่อเรียงดอกไม้ชนิดเดียวกันครบ 5 ช่อง

## 🎮 วิธีการเล่น (How to Play)
1. **ย้ายดอกไม้:** คลิกหรือแตะที่ดอกไม้ที่ต้องการ จากนั้นเลือกช่องว่างที่ต้องการย้ายไป
2. **การทำคะแนน:** เรียงดอกไม้ชนิดเดียวกัน (หรือใช้ดอกไม้รุ้ง 🌈 ช่วย) ให้ครบ 5 ช่องในแนวตั้งหรือแนวนอน
3. **ความท้าทาย:** ทุกครั้งที่ย้ายดอกไม้แล้วไม่เกิดการบาน ดอกไม้ใหม่จะถูกสุ่มขึ้นมา 3 ดอก หากสวนเต็มจะจบเกมทันที

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **Language:** ES6+ JavaScript (Logic & Game Mechanics)
- **Styling:** CSS3 (Flexbox, Grid, Keyframe Animations)
- **Markup:** HTML5
- **Platform:** GitHub Pages (Deployment)
